# CRE Investment Analysis Platform

**Live site →** https://srikargadireddy.github.io/CRE-calculator-/

A world-class, single-page CRE deal analysis tool. Upload an OM, T12, or Rent Roll (PDF / Excel / CSV) and get instant, AI-powered deal insights across 11 analysis modules.

## Features

- **11-module left sidebar** — Dashboard · Returns · Operations · Leasing · Unit Economics · Debt Structure · Valuation & Exit · Market Analysis · ESG · Due Diligence · Ask Document
- **Animated landing page** — Motion One-powered orbs, CSS star-field, shimmer title, glow upload zone
- **Live Scenario Modeler** — sliders drive real-time DSCR, cap rate, exit value and CoC calculations
- **60+ CRE attributes** auto-populated from uploaded documents
- **Ask Document chat** — quick-question buttons + free-form query with Anthropic API support

## Tech stack

Vanilla HTML / CSS / JavaScript — no build step required.
Animations powered by [Motion One](https://motion.dev) (`motion@10.18.0`).
